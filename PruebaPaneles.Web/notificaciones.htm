Muestra el listado de notificaiones que contiene el usuario.

Las notificaciones podrán ser recibidas por el sistema debido a una modificación realizada sobre algún proyecto o por algún usuario conectado para enviarle algún mensaje.

Además permite eliminar una o varias notificaciones y crear una nueva notificación para enviar a un usuario conectado.
